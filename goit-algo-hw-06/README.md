ВИСНОВКИ:
================================================================================

DFS (Depth-First Search):
  • Йде вглиб графа, досліджуючи одну гілку до кінця
  • НЕ гарантує найкоротший шлях
  • Використовує стек (LIFO - Last In First Out)
  • Може знайти довший шлях через глибокий пошук

BFS (Breadth-First Search):
  • Досліджує граф пошарово, рівень за рівнем
  • ГАРАНТУЄ найкоротший шлях (мінімальна кількість станцій)
  • Використовує чергу (FIFO - First In First Out)
  • Завжди знаходить оптимальний маршрут для незваженого графа

Для транспортних мереж краще використовувати BFS
